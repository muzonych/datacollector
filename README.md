# 🤖 Smart Prospector: AI Lead Magnet Bot

A high-conversion Telegram bot designed to replace static contact forms and qualify leads using AI logic.

[📺 CLICK HERE TO WATCH THE DEMO](https://youtube.com/shorts/6ZrQ4MaouWc?si=mCh_I1T1E1Q0Cao7)

## ✨ Features
- **Interactive Onboarding:** Engages users with a natural conversation flow.
- **FSM Architecture:** Robust state management for error-free data collection.
- **Instant Admin Alerts:** Real-time notifications for every new lead captured.
- **Native Experience:** Built with `aiogram 3.x` for maximum speed and reliability.

## 🚀 Quick Start
1. Clone the repo: `git clone [url]`
2. Install dependencies: `pip install -r requirements.txt`
3. Add your `API_TOKEN` and `ADMIN_ID`.
4. Run: `python main.py`

## 💼 Business Use Case
Perfect for real estate agents, online coaches, and service-based businesses. This bot automates the "first touch" and ensures no lead is lost.
